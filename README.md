# WeatherGUI

Simple Python GUI application to display weather. Created using Tkinter.

## Requirements

- Python 3.8

## Installation

```bash
cd weather-gui
python -m venv venv

# Activate virtual env

# Windows
venv\Scripts\activate

# Linux
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
```

## Usage

```bash
cd weather-gui/src
python main.py
```
